# Personal Website

**URL:** https://rczheng.com

## Summary

Personal portfolio website for Ruichen Zheng, featuring bio, CV, current projects, and travel portfolio. Includes orientation-responsive layouts (landscape/portrait) with smooth scrolling and theme switching.

## Technologies Used

- **Next.js** (v16.1.1) - React framework
- **React** (v19.2.3) - UI library
- **TypeScript** - Type-safe development
- **Tailwind CSS** (v3.4.4) - Styling
- **Node.js** - Runtime environment
- **Vercel** - Deployment platform
- **Remark** - Markdown processing
- **Claude Code** - Development assistance

## Development

```bash
npm install
npm run dev
```
